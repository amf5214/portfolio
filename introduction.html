Throughout my time working at Amazon.com, the Financial Independence Group, and my education I have many projects both public and confidential dealing with large scale customer facing products. My publicly available projects can be found on my github page.

  • Station Assignment Tool: The station assignment tool is a fully built out java application running spring boot as the backend webinterface that worked to staff over 8,000 associates across 14 shifts and 3 departments that intelligently predicted where associates needed to be placed based off of 12 differnt internal KPIs.

  • ARC: Large scale customer facing tool used to create and manage customer contracts for over 2 million customer transactions focusing in annuities and insurance built primarily in the .NET framework and managed by a team of 14 software developers.

  • TyeDyeInk.com: Customer facing web application currently in development to service a new and growing business, TyeDyeInk, allowing customers to buy custom products and interact with the designer of the clothing.
